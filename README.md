# Integrated-Financial-Model-Excel
Integrated 3-statement financial model built in Excel including income statement, balance sheet, cash flow statement, and supporting schedules

## Overview
This project is an Excel-based financial model integrating the three core financial statements:

- Income Statement  
- Balance Sheet  
- Cash Flow Statement  

## Key Features
- Revenue forecasting using growth assumptions  
- Expense modeling using % of sales  
- Working capital analysis (AR, Inventory, AP)  
- Cash flow modeling (Operating, Investing, Financing)  
- Debt schedule (Line of Credit & Long-Term Debt)  
- PP&E schedule with depreciation  

## Model Highlights
- Fully integrated 3-statement model  
- Cash flow dynamically drives ending cash balance  
- Balance sheet validated (Check = 0)  
- Includes assumptions and financial projections  

## Purpose
This model was built as part of a financial modeling learning exercise based on a tutorial. The objective was to develop a practical understanding of:

- Financial statement integration  
- Excel-based financial modeling  
- Corporate finance concepts  

## Tools Used
- Microsoft Excel  
- Financial modeling techniques
